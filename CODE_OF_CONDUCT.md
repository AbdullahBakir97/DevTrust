# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as its Code of Conduct.

The full text of the Covenant is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: we expect everyone participating in this project — through issues,
pull requests, discussions, or any other channel — to act with respect,
patience, and good faith toward other contributors. Disagreement is welcome.
Personal attacks, harassment, and dismissive behavior are not.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please report it to the project maintainers at:

> conduct@devtrust.dev *(placeholder until the org is registered; until then,
> open a private GitHub Security Advisory on the repository instead)*

Reports are reviewed by the maintainers and responded to within 5 business
days. Reporters' identities are kept confidential.

## Enforcement

Maintainers will follow the Contributor Covenant's enforcement guidelines
(see the URL above) when responding to reported behavior. Possible responses
range from a private warning to a temporary or permanent ban from the project.

## Scope

This Code of Conduct applies to all project spaces — issue trackers, pull
requests, discussions, chat, and any official off-platform venue (conference
talks, meetups) where the project is represented.
